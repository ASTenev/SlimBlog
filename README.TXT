Mini Blog App with Slim 3.0

This is a mini blog app built with Slim 3.0, a lightweight PHP micro-framework, that allows users to register, log in, create posts, and delete posts. The app requires PHP 7.3 or higher and MySQL 5.7.

INSTALLATION
To install the app, follow these steps:

1.Clone the project repository:
git clone https://github.com/[username]/mini-blog-app.git

2.Install the project dependencies using Composer. Run:
    composer install

3.Configure the app settings MySQL settings in .env file

4.Create manualy database in MySQL.

5.Run database migration with the folowing command:
    vendor/bin/phinx migrate

6.Open in browser and enjoy!
